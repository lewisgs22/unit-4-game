Crystal Collector!

This is a game in which there is a randomly generated goal number between 19-120  that you try to match by clicking on one of four gems.
If you go over the goal number you lose.
The gems themselves represent a randomly generated number between 1-12 that resets when you either win or lose.
Wins and losses are both tallied over as many games as you like.
Game is reset without reloading the page.
